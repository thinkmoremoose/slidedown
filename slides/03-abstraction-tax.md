## The Abstraction Tax
<p class="muted">Every framework demands AI learn:</p>
* 📦 A proprietary file format (OOXML, Protobuf, JSON blobs…)[[fragment]]
* 🔌 A unique API surface — hundreds of endpoints[[fragment]]
* 🧩 A plugin and extension ecosystem[[fragment]]
* 🔄 Version-specific quirks and breaking changes[[fragment]]
* 📚 Thousands of pages of documentation — per tool[[fragment]]

<p class="fragment callout">More effort adapting to the tool than creating value.</p>

<aside class="notes">PowerPoint alone has over 300 distinct object types in its API. Google Slides has a completely different object model. Keynote barely has an API at all. Every framework is a walled garden with its own rules, and AI has to learn them all — separately.</aside>
