## Zero Dependencies
* 🚫 No install, no license, no subscription[[fragment]]
* 💻 Works on every OS — if it has a browser, it works[[fragment]]
* ✈️ Works offline — it's just a file on your disk[[fragment]]
* 🏛️ Will still work in 20 years — HTML is eternal[[fragment]]
* 📦 This entire presentation: <strong class="accent">&lt; 30 KB</strong>[[fragment]]

<p class="fragment muted" style="margin-top: 24px; font-size: 26px;">Compare: PowerPoint requires a 4 GB install and $160/year.</p>

<aside class="notes">The durability point is key. HTML files from 1995 still render correctly in modern browsers. Can you open a PowerPoint 95 file today? Maybe — with compatibility warnings, broken fonts, and missing features. HTML just works. Forever.</aside>
