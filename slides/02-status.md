## The Current Approach
<p class="muted">How we teach AI to make presentations today:</p>
* 🔧 Learn PowerPoint's COM API and OOXML format[[fragment]]
* 🔧 Learn Google Slides' REST API and OAuth flow[[fragment]]
* 🔧 Learn Keynote's AppleScript interface[[fragment]]
* 🔧 Then Canva, Prezi, Figma, Pitch…[[fragment]]

<p class="fragment callout">Each framework is a new language to master.<br>Each update is a new way to break.</p>

<aside class="notes">The current paradigm is teaching AI to operate existing tools — essentially building AI-powered wrappers around legacy software. Every time a framework updates, every integration risks breaking. We're on a treadmill, and the treadmill keeps accelerating.</aside>
