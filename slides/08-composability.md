## The Web Platform <span class="accent">IS</span> the Framework
<p class="muted">Need something? Just add it.</p>
* 📊 Need a chart? → <code>&lt;canvas&gt;</code> or <code>&lt;svg&gt;</code>[[fragment]]
* ❓ Need a quiz? → 20 lines of JavaScript[[fragment]]
* 🎮 Need 3D? → WebGL, right in the browser[[fragment]]
* 📡 Need live data? → <code>fetch()</code> API[[fragment]]
* ⏱️ Need a timer? → <code>setInterval()</code>[[fragment]]
* 🎵 Need audio? → <code>&lt;audio&gt;</code> element[[fragment]]

<p class="fragment callout" style="border-left-color: var(--accent); background: rgba(137,180,250,0.06);">Try embedding a live API call in a PowerPoint slide.</p>

<aside class="notes">The browser is the most capable application platform ever built. Every year it gets more powerful. PowerPoint hasn't fundamentally changed in two decades. When you build directly for the browser, you get access to the entire modern web platform — for free.</aside>
