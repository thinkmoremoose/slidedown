## But What About…?
* <strong>"Collaboration?"</strong> <span class="muted">→ Git + any text editor. Better version control than Google Slides.</span>[[fragment]]
* <strong>"Non-technical users?"</strong> <span class="muted">→ AI generates the file for them. That's the whole point.</span>[[fragment]]
* <strong>"Brand templates?"</strong> <span class="muted">→ CSS custom properties. Swap one variable block.</span>[[fragment]]
* <strong>"Editing in place?"</strong> <span class="muted">→ That's just another app to build.</span>[[fragment]]
* <strong>"Existing content?"</strong> <span class="muted">→ AI can convert formats too.</span>[[fragment]]

<p class="fragment callout">None of these are blockers. They're all solvable — and most are already easier than the PowerPoint equivalents.</p>

<aside class="notes">The collaboration objection is the most common, but think about it: Google Docs solved collaboration not by teaching AI to use Word's COM API, but by building a new purpose-built app. The same logic applies here.</aside>
